# Extra files for ObiWanStenobit keyboard

This directory contains supplementary files related to the ObiWanStenobit keyboard development.  
These files are not required to build or flash the firmware, but they are useful for layout design, visualization, and tooling during development.

The following file types are included:

- `KLE-*.json`  
  Files for [Keyboard Layout Editor](https://www.keyboard-layout-editor.com/)

- `KLT-*.json`  
  Files for [Keyboard Layout Tools](https://nickcoutsos.github.io/keymap-layout-tools/)

>Not all files in this directory are guaranteed to be up to date with every firmware release.
